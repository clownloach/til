# Google Maps API

地理的データを位置情報として提供する。

* Maps Statc API

## Maps Static API

埋め込み可能な地図画像を取得する。(例.https://maps.googleapis.com/maps/api/staticmap?key={APIキー}&center=37.62761,-122.42588&zoom=17&format=png&sensor=false&size=640x480&maptype=roadmap)

### 導入

1. APIライブラリ -> Maps Static APIを選択。
2. 有効にするをクリック。
